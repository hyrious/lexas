Single-page application Playground

  • OPG parser

编译原理作业（1）使用 OPG parser 解析类似 9-5+4 的字符串，生成 AST。

使用到的外部库：Ace 编辑器本体；MDUI 显示一个小提示框。

OPG parser 的实现在该页源码里，使用 es6 编写，不支持过老的浏览器和 IE。

  • Markdown to Discuz!bbcode

将 markdown 转成 Discuz! 论坛使用的 bbcode 代码。

使用到的外部库：Ace 编辑器本体；marked 解析 markdown。

  • Pixiv profile banner generator

生成带有 pixiv 用户 ID 的图标，尝试将地址栏的 ?id=23333 改成其他数字。

使用到的外部库：Qs 解析地址栏字符串。

  • QRCode Helper

二维码生成/解析。

使用到的外部库：qrcodejs 生成二维码；qrcode-reader 解析二维码；Primer CSS 界面；Qs 解析地址栏字符串。
